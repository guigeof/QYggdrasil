## 4. 📋 Workspaces & QGIS Sync
Workspaces allow you to manage project-specific collections and synchronize them directly with your desktop QGIS environment.

![Workspaces Management](file:///C:/Users/guilh/.gemini/antigravity/brain/cd76312f-c274-4615-930f-8d32305a8811/workspaces_1775489243468.png)

*   **Custom Lists**: Create lists for different workflows (e.g., "Hydrological Project", "Cartography Pack").
*   **Status Management**: Toggle plugins within your list to plan your installation.
*   **⚡ Ghost Sync**: Export your workspace as a `.json` file to be consumed by the **QYggdrasil Desktop Companion**.
*   **Python Snippet Generator**: Copy-paste a Python snippet directly into the QGIS Python Console for instant batch installation.

---

## 🔌 Desktop Companion Installation
1.  Navigate to the **Workspaces** tab.
2.  Click **Download ZIP** for the Desktop Companion.
3.  In QGIS, go to **Plugins** > **Manage and Install Plugins** > **Install from ZIP**.
4.  Import your exported `.json` Workspace to sync all your selected plugins in seconds.
