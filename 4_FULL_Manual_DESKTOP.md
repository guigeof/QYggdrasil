# 🔌 QYggdrasil: Desktop Companion Manual
## Bridge discovery to deployment directly inside QGIS

The QYggdrasil Desktop Companion is a native QGIS plugin that allows you to synchronize your carefully curated workspaces from the Web Hub into your local GIS environment.

---

## 🛠️ 1. Installation

Since this plugin is a companion tool, it is installed manually from a ZIP file generated by the Web Hub.

1.  **Download**: In the QYggdrasil Web Hub, navigate to the **Workspaces** tab and click **Download Desktop Companion**.
2.  **Plugin Manager**: Open QGIS and go to `Plugins` > `Manage and Install Plugins...`.
3.  **Install from ZIP**: 
    *   Select the **Install from ZIP** tab on the left.
    *   Browse to the downloaded `workspace_manager.zip`.
    *   Click **Install Plugin**.
4.  **Activation**: Ensure the "Workspace Manager" is checked in your installed plugins list.

---

## 🔄 2. Synchronizing Your Workspace

There are two primary ways to sync your discoveries from the web to your desk:

### Method A: JSON Import (Recommended)
1.  **Export**: In the Web Hub, click **Export Workspace (JSON)** on any of your plugin lists.
2.  **Open Plugin**: In QGIS, locate the QYggdrasil icon in your toolbar or under the `Plugins` menu.
3.  **Load JSON**: Click the **Import Workspace** button and select your `.json` file.
4.  **Apply**: The plugin will now list all plugins from your web session.

### Method B: Python Snippet
1.  **Copy**: In the Web Hub Workspace tab, click **Copy QGIS Python Snippet**.
2.  **Console**: In QGIS, open the Python Console (`Ctrl+Alt+P`).
3.  **Run**: Paste the code and hit `Enter`. QGIS will automatically begin downloading and enabling every plugin in your list.

---

## ⚡ 3. Managing Workflows
The companion plugin allows you to treat groups of plugins as "Workflows."

*   **Batch Enable/Disable**: Quickly swap between "Hydrology" and "Cartography" toolkits by toggling your workspace.
*   **Missing Plugin Detection**: The plugin automatically scans your system; any items in your workspace that aren't installed will be highlighted for one-click installation.
