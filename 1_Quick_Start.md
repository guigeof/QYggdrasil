# ⚡ QYggdrasil: Quick Start Guide

A high-density **Instrument Panel** workstation for the QGIS Plugin Ecosystem.

## 🏁 Fast Setup

1. **Web Hub**: Open `http://localhost:8000` (or the GitHub Pages link) to access the professional GIS dashboard.
2. **Desktop Plugin**: Navigate to the **Workspaces** tab, download the `workspace_manager.zip`, and install it in QGIS via **Plugins > Manage > Install from ZIP**.

## 🛠️ core Workflow

| Step                  | Action                                                                   | Outcome                |
| :-------------------- | :----------------------------------------------------------------------- | :--------------------- |
| **1. Discover** | Use the**Taxonomy Tree** (Yggdrasil) to find thematic plugins.     | Contextual discovery.  |
| **2. Explore**  | Use the**Nano-Explorer** grid to compare versions and ratings.     | Technical assessment.  |
| **3. Plan**     | Add plugins to a**Workspace** (Hydrology, Cartography, etc.).      | Project organization. |
| **4. Sync**     | Export Workspace as `.json` or copy the **QGIS Python Snippet**. | One-click deployment.  |

## 📐 Keyboard/Visual Shortcuts

* **Grid Column Toggle**: The Explorer grid adjusts layout for maximum density based on your screen size.
* **Modal Esc**: Tap `ESC` to quickly close plugin detail cards.
* **Theme**: The **Matte Graphite** UI is permanently active for zero-fatigue professional use.
